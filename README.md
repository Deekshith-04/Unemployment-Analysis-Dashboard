# Unemployment Analysis Dashboard

An interactive data analytics dashboard built using **Python, Streamlit, and Machine Learning** to analyze unemployment trends in India and predict unemployment rates.

---

## Live Demo
https://unemployment-analysis-dashboard-04.streamlit.app/
---

## Features

- Unemployment trend analysis (state-wise & area-wise)
- Rural vs Urban comparison
- Top 10 states with highest unemployment
- COVID-19 impact visualization
- Machine Learning model for unemployment prediction
- Interactive filters (State & Area selection)
- Clean dashboard-style UI

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

---

## Dataset

- Unemployment in India dataset
- COVID period unemployment dataset

---

## Machine Learning

- Model: Linear Regression
- Features:
  - Estimated Employed
  - Labour Participation Rate
- Target:
  - Unemployment Rate
